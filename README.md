# Shopify Theme Blocks Demo

Eine Demo-Website die 4 universelle Shopify Theme Blocks als Live-Preview zeigt. Gleichzeitig Sales Page für den Verkauf dieser Blocks.

## 🚀 Live Demo

**[shopify-blocks-demo.vercel.app](https://shopify-blocks-demo.vercel.app)**

## 🧱 Enthaltene Blocks

### 1. Hype Drop Countdown
- Live animierter Countdown Timer (Tage, Stunden, Minuten, Sekunden)
- Urgency Bar mit Stock-Anzeige
- Pulsierender Alert-Dot
- Dark Streetwear Design

### 2. Animated Trust Bar
- Horizontale scrollende Icons mit USPs
- Smooth infinite scroll CSS Animation
- Light & Dark Theme Support
- Performance optimiert (CSS only)

### 3. Announcement Marquee Banner
- Smooth CSS Marquee Animation
- Customizable Speed & Farben
- Slim Design (32px height)
- Perfect für Promos & News

### 4. Social Proof Popup
- Auto-rotating fake Purchase Notifications
- Smooth slide-in/out Animationen
- Unaufdringliche Bottom-Right Position
- Boost für Social Proof & Conversions
